# SafeClicks

SafeClicks is an Android application designed to protect women and children from phishing and malicious links by automatically scanning shared URLs in real time.

## Features
- Real-time phishing detection
- Parental controls
- Community-based reporting
- User-friendly interface

## Technology Stack
- Android (Java/Kotlin)
- Firebase / API (if used)

## Purpose
This project aims to improve digital safety and prevent cyber fraud.
